# Liquor Shop Landing Page

This is a landing page for a premium liquor shop, built with Next.js and Tailwind CSS.

## Features

- Responsive design
- Product showcase
- "Why Choose Us" section
- Store locations
- About section
- Contact information

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Technologies Used

- Next.js
- React
- Tailwind CSS
- Lucide React Icons
